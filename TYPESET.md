# TYPESET (/paper/)

Typesetting specification for PAPER PDF output.

---

## Purpose

This specification defines the implementation details for rendering PAPER as PDF.
TYPESET is descriptive and non-authoritative; governance is defined exclusively
by CANON.

---

## Layout

### Two-column layout

The manuscript body uses a two-column layout.

### Appendix ordering

Appendices follow the manuscript in declared order.

---

## Figures

### Vector rendering

Figures render as vector graphics (mermaid or SVG).

### Font sizing

Figure text matches the manuscript body font size.

### Column spanning

Figures may span both columns when required for legibility.

### Asset embedding

Figures are embedded as LaTeX figures using assets from `paper/figures/`.

### Mermaid sources

Mermaid sources are rendered to SVG before embedding.

---

## Tables

### Evidence tables

Long evidence tables live in appendices to preserve narrative flow.

---

## Captions

### Figure captions

Figure captions follow the figure.

### Table captions

Table captions precede the table.

---

*This specification is descriptive and non-authoritative. Governance is defined
exclusively by CANON.*
