## Purpose

LEARNED defines derived artifacts that reference source episodes and ledger references. Learned artifacts must record channel and source references, including transcript record IDs when available. Learned artifacts must not include transcript content.

---

## Scope

### In scope

- Derived artifact creation
- Episode and ledger referencing
- Channel and source disclosure
- Transcript record ID tracking

### Out of scope

- Transcript content storage
- Primary episode modification
